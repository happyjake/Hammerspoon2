    Namespace http

    HTTP client module for making network requests from JavaScript. All request methods return Promises that resolve with a result object containing status (number), body (string), and headers (object). On network failure, status is -1 and body is an empty string.

    Quick start

    hs.http.get("https://api.example.com/data").then(r => {
       if (r.status === 200) {
           console.log("Got: " + r.body)
       }
    })
