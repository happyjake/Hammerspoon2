    Class HSBLEPeripheral

    A handle to the relay peripheral on the bonded ESP32. Returned by HSBLECentral.connect(...). Register onConnect / onDisconnect / onNotify, and write lines to the controller. onConnect fires once the relay notify characteristic is subscribed (the channel is live).
