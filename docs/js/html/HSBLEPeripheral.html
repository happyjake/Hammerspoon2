HSBLEPeripheral

Type

A handle to the relay peripheral on the bonded ESP32.

Returned by HSBLECentral.connect(...). Register onConnect / onDisconnect / onNotify, and write lines to the controller. onConnect fires once the relay notify characteristic is subscribed (the channel is live).

Properties

uuid

string
The peripheral's system UUID. Persist this (e.g. in your config) and pass it back as `connect({ peerUUID })` for instant re-attach to the non-advertising bonded peer. Empty until first connected.
