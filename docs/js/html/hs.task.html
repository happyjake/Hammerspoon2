hs.task.runSync(launchPath, arguments) -> string

Run a short-lived command synchronously and return its stdout as a string. Use sparingly — this blocks the JS thread until the process exits. Intended for fast utilities (`ps`, `whoami`, `uname`) where awaiting a Promise would add UI flicker.
Declaration
hs.task.runSync(launchPath, arguments) -> string
Parameters
Name Type Description
launchPath string Absolute path to the executable
arguments string[] Argument array
Returns
string
Combined stdout as a string, or null on failure
Example
const out = hs.task.runSync('/bin/ps', ['-axo', 'pid,rss,comm'])
