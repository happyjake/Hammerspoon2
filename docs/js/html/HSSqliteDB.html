HSSqliteDB

Type

A per-connection SQLite database object returned by hs.sqlite.open(). Wraps a sqlite3 handle and exposes synchronous exec, parameterized run/query, and transaction helpers to JavaScript.

transaction(fn) -> JSValue

Run a JS function inside a BEGIN/COMMIT pair. If the function throws, the transaction is rolled back and the exception is re-thrown to the caller. Returns the function's return value on success. Nested transactions throw — savepoints are not supported in v1.
Declaration
transaction(fn) -> JSValue
Parameters
Name Type Description
fn JSValue A function with no arguments
Returns
JSValue
The function's return value, or null on rollback
Example
db.transaction(() => {
  db.run('UPDATE t SET name = ? WHERE id = ?', ['bob', 1])
  db.run('DELETE FROM t WHERE id = ?', [2])
})
