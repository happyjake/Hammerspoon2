HSMPCSession

Type

A MultipeerConnectivity session — the CrossMac data plane.

Obtain via hs.multipeer.session(...). Call start() to advertise + browse; both peers advertise and browse, and whichever sees the other first invites, authenticated by the shared context string. Payloads cross the JS bridge as base64 strings (pairs with HSImage.encode).

Properties

peers

string[]
The display names of all currently connected peers.
