hs.mouse.position() -> {[key: string]: number}

The current mouse cursor position in global screen coordinates (top-left origin, points). Coordinates match `hs.screen` convention: `(0,0)` is the top-left of the primary display and `y` increases downward.
Declaration
hs.mouse.position() -> {[key: string]: number}
Returns
{[key: string]: number}
An object `{ x, y }`.
Example
const p = hs.mouse.position(); console.log(p.x, p.y)

hs.mouse.setAssociated(connected) -> boolean

Connect or disconnect physical mouse movement from the on-screen cursor. Pass `false` to decouple movement from the cursor position (useful for relative-delta capture, e.g. seamless hand-off to a remote machine).
Declaration
