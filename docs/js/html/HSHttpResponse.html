HSHttpResponse

Type

One outgoing HTTP response. Models the WHATWG Fetch Response shape: status, statusText, headers, and a body-as-bytes accessor. Returned by the user's fetch handler.

Properties

status

number
HTTP status code (e.g. 200, 404).
Hammerspoon 2/Modules/hs.httpserver/HSHttpResponse.swift:25

statusText

string
HTTP status text. Defaults from `status` per RFC 7231 if not provided.
