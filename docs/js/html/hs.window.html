hs.window.snapshot() -> [[String: Any]]

Get a snapshot of the live window registry — apps and their windows in MRU order, populated from observers. Reads from cache; no AX calls on the hot path. Use this in latency-sensitive code like switchers.
Declaration
hs.window.snapshot() -> [[String: Any]]
Returns
[[String: Any]]
An array of dictionaries: `[{pid, name, bundleID, iconBase64, windows: [{id, title}]}]`
Example
const snap = hs.window.snapshot()
console.log(snap[0].name, snap[0].windows.length)
