hs.httpserver

Module

Multi-map of HTTP header name → value(s), with case-insensitive lookup per RFC 7230 §3.2. Backs both incoming HSHttpRequest.headers and outgoing HSHttpResponse.headers. Mirrors the WHATWG Fetch Headers API: get, set, append, delete, has, iteration.

Types

This module provides the following types:

  • HSHttpHeaders
  • HSHttpRequest
  • HSHttpResponse
  • HSHttpServer

Properties

This module has no properties.
